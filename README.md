# Portfolio
Portforlio website project

## Getting Started
### Dependencies
* ReactJs
* Gatsby
* React Bootstrap
* Bootstrap
* styled components

### Template
Creative Startbootstrap https://startbootstrap.com/themes/creative/

### Installing
```
npm install -g gatsby-cli
```
```
npm install
```

### Executing program

```
gatsby develop
```
```
gatsby build
```
